Waipoua Changelog:

Version 1.0.4 - August 7th 2012
----------------------------------
1. New Twitter icon image (see images/socialicons.png and images/2x/twitter.png)
2. Included a Dutch translation (see folder languages, nl_NL.mo + nl_NL.po)
3. Optimized Media Queries for HiDPI (Retina) displays (see style.css, 3001)
4. Bugfix for image aligns in admin text editor (see editor-style.css, 226)
5. Adding padding for caption text on blog front page (see style.css, 673)
6. Bugfix for border-bottom lines in sub menus (see style.css, 2866-77)
7. Optimized text styling for info boxes (see style.css, 1415-22)
8. Deleted hover text-underline in footer (see style.css, 2000-05)


Version 1.0.3 - July 10th 2012
----------------------------------
1. Google Webfont bugfix: Deleted the Google Webfonts @import in Stylesheet (in style.css, 14).
2. Included a Swedish theme translation (see languages folder).


Version 1.0.2 - July 7th 2012
----------------------------------
1. Included Google Webfonts via functions.php instead of @import in style.css (see functions.php, 89)
2. Bugfix for Retina-display optimized Github icon (see style.css, 3113)
3. Bugfix for overlapping dropdown menus, added z-index: 99999; (see style.css, 2288 + 2821)
4. Added "Edit Post/Page" link for single post pages and pages (see content-page.php, 27 and content-single.php, 16)
5. Bugfix for font-size inside shortcode info boxes (see style.css, 1420-1431)
6. Fixed standard content-width size (see functions.php, 26)
7. Improved image attachment styles (see style.css, from 1361 + image.php 58-74)
8. Additional translations for the German language files (see folder languages, de_DE.mo + de_DE.po)


Version 1.0.1 - June 6th 2012
----------------------------------
1. Added Google Webfonts PT Sans 400 Italic font style (see style.css, 18).
2. Included backward compatibility for the custom backgrounds function (see functions.php, 78-84)
3. Added optimized styling for img.wp-post-image for mobile devices (see style.css, 622 + 2520)
